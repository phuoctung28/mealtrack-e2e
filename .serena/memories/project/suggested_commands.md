# Suggested commands (mealtrack-e2e)

## Install

```bash
npm ci
```

## Run tests

```bash
npm test
npm run test:e2e
```

## Lint / format

```bash
npm run lint
npm run format
```

## Required env vars (local)

- `E2E_BASE_URL`
- `FIREBASE_WEB_API_KEY`
- `FIREBASE_SERVICE_ACCOUNT_JSON`
- `E2E_UID` (optional)
